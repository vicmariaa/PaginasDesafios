Página 2 – Aplicação Web com JavaScript

Calculadora de IMC

Como Funciona

O HTML fornece os campos e o botão. O JavaScript faz o cálculo do IMC e mostra o resultado na própria página.

Explicação do Código

  • HTML: Cria a interface com campos de entrada e botão.
  • JavaScript: Lê os dados, calcula o IMC e atualiza a tela.
  • Tudo no Frontend: Nenhum dado vai para o servidor.

Vantagem

Esse tipo de aplicação é simples e roda direto no navegador. Ideal para ferramentas rápidas e interativas.

Comparação com Python

Python no backend (como mostrado na Página 1) exige um servidor rodando, mas permite interações mais complexas e acesso a banco de dados. Já o JavaScript com HTML roda sozinho no navegador, sem backend.
